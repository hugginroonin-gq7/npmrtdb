# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-31

### Added
- Initial release
- `npmrtdb` - NPM wrapper with multi-host support
- `npmxrtdb` - NPX wrapper with multi-host support
- Concurrent execution across multiple registries
- Public-first strategy with token fallback
- Two version selection modes: `any` and `latest`
- Digit-based version comparison for `latest` mode
- Cross-platform support (Windows, Linux, macOS)
- Debug mode with detailed logging
- JSON output option
- Comprehensive error handling
- Automatic cleanup of temporary files
- Pass-through support for all npm/npx flags
